⚙️ Eenmalige instelling

Verbind de app met jouw gratis Supabase database. Dit hoef je maar één keer te doen — de gegevens worden opgeslagen in je browser.

📋 Stap 1 — Maak een gratis Supabase project

  1. Ga naar supabase.com → Start your project (gratis)
  2. Maak een nieuw project aan (kies een regio dicht bij Nederland)
  3. Ga naar Project Settings → API
  4. Kopieer de Project URL en de anon / public key

🗄️ Stap 2 — Maak de tabel aan

  1. Ga in Supabase naar SQL Editor
  2. Klik New query en plak onderstaande SQL:
create table if not exists measurements (
  id           bigserial primary key,
  name         text not null,
  behaviour    text not null,
  pros         smallint not null,
  cons         smallint not null,
  self_eff     smallint not null,
  support      smallint not null,
  measured_at  timestamptz not null default now()
);
alter table measurements enable row level security;
create policy "insert_open" on measurements
  for insert with check (true);
create policy "select_open" on measurements
  for select using (true);

Klik daarna op Run in Supabase.

🔑 Stap 3 — Vul je gegevens in

Te vinden onder Project Settings → API → Project URL
Te vinden onder Project Settings → API → Project API keys → anon public
Stap 1 van 3

Welkom bij CoachingEffect

Deze app helpt jou en je coach inzicht te krijgen in hoe jouw motivatie en zelfvertrouwen zich ontwikkelen rondom een specifiek gedrag. Je kunt zo vaak meten als je wilt — dagelijks of vaker. De resultaten worden overzichtelijk in een grafiek weergegeven.

Stap 2 van 3

Wat ga jij doen?

Beschrijf zo concreet mogelijk welk gedrag je wilt monitoren. Specifiek betekent: wat, hoe vaak, wanneer.

3× per week 30 min wandelen Minder vet eten door de week Geen alcohol door de week
